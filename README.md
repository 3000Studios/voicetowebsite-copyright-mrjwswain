# VoiceToWebsite

Static + Vite-powered site for the VoiceToWebsite ecosystem (public pages + admin tools + app store).

## Local dev

- Install: `npm install`
- Run: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
