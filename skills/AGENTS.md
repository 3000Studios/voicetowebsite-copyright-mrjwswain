# AGENTS.md (skills/)

## Scope

- Applies to files under `skills/`.
- Inherits root rules from `/AGENTS.md`.

## Skill authoring rules

- Keep each skill focused on one task family.
- Include:
  - `Use when`
  - clear step sequence
  - `Exit criteria`
- Use real repo paths and command names only.
- Update `skills/README.md` when adding/removing skills.
