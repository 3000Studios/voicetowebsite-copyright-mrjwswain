# VoiceToWebsite Full-Spectrum Generator Law

This document is mandatory project law for VoiceToWebsite.com.

## Canonical Project Identity

- The only repository for anything associated with VoiceToWebsite.com is `3000Studios/voicetowebsite-copyright-mrjwswain`.
- The only production branch is `main`.
- `main` is the live branch.
- The only deployment path is Cloudflare Wrangler.
- GitHub deployment and GitHub Actions production deployment are not allowed.
- The only production domains are `voicetowebsite.com` and `www.voicetowebsite.com`.
- No legacy deployment provider deployment, legacy deployment provider configuration, legacy deployment provider route, or legacy deployment provider branding is allowed.
- No voicetowebsite identity, copy, project name, deployment target, route, storage key, or branding is allowed.

## Mission

Build a voice-controlled cinematic website, logo, dashboard, and brand-kit factory that generates custom animated digital businesses with live previews, endless variations, premium hover effects, responsive wallpapers, modals, SEO, monetization, mobile-first layouts, history, export, upgrade gates, and preview-only watermark protection without paid API fees and with Cloudflare Wrangler as the deployment path.

## Mandatory Product Direction

The system must generate complete, premium, cinematic, animated, responsive, accessible, monetized, SEO-ready, production-ready websites, dashboards, logos, brand kits, live previews, editor systems, and exportable assets from short voice or text prompts.

The generator must never create boring, generic, plain, weak, unfinished, placeholder-heavy, or broken output.

The generator must work as a free local cinematic composition engine by default. It must not require paid AI API keys for the core generator, website preview, logo preview, dashboard/editor, or publishing workflow.

## Core Generator Behavior

The generator must:

- Accept voice prompts and typed prompts.
- Understand the business, industry, target audience, business goal, monetization path, requested tone, requested style, and location when provided.
- Create a full site blueprint before rendering.
- Choose a cinematic design direction and unique layout system.
- Generate custom business-specific content.
- Generate animated backgrounds, hover effects, motion effects, responsive mobile/tablet/desktop layouts, modals, CTA buttons, forms, trust sections, preview watermark, variation generation, random generation, publishing, upgrade paths, scoring, saving, remixing, and regeneration.
- Feel like a live design engine, not a template form.
- Generate branded, animated, monetized, conversion-ready digital businesses.
