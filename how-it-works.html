The flow

Fast preview for buyers. Safe apply for operators.

1

2

Generate structure

AI outputs a multi-page layout: nav, pages, and 2–5 sections per page.

3

Preview instantly

4

Admin-only sandbox apply

Voice commands can update the sandbox page safely without touching production pages.

5

Publish + iterate

Publish generated sites to R2 (admin) and keep improving with Plan → Apply gates.

What “sandbox” means

Safe target file

Admin orchestration can be locked to `sandbox.html` only.

Supported actions

Update headline/subhead/CTA, theme, insert sections, swap background video.

No secrets exposed

Never inject passwords or signing keys into client-side `window.__ENV`.
